Getting Started

Installation

To install FieldFrame, follow these steps:

  1. Download and unzip the latest version
  2. Upload extensions/ext.fieldframe.php and the extensions/fieldtypes folder to system/extensions
  3. Upload language/english/lang.fieldframe.php to system/language/english
  4. If you’re using FieldFrame fieldtypes in a SAEF, upload themes/cp_themes/default/ff-saef.css to themes/cp_themes/default
  5. Enable FieldFrame in the Extensions Manager

Now you need to tell FieldFrame where your fieldtypes folder lives. Go to FieldFrame’s Settings, and enter the Fieldtypes Folder Path and the Fieldtypes Folder URL. Click Submit, and then go back to FieldFrame’s Settings. If you entered the Fieldtypes Folder Path correctly, your Fieldtypes Manager will get populated with FF Checkbox, FF Checkbox Group, etc.

Managing Your Fieldtypes

Much like the Extensions Manager, the Fieldtype Manager (in FieldFrame’s Settings) lists out all of your installed fieldtypes. Activate a fieldtype by selecting the radio button labelled “Yes” in the “Enabled?” column.

If a fieldtype has its own site-wide settings, they won’t be visible until you come back to FieldFrame’s Settings after the fieldtype has been activated. Click the “Show” link in the Settings column of the Fieldtypes Manager to reveal them.
