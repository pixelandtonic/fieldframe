Overview

Requirements

In order to get FieldFrame up-and-running, you will need:

  • PHP 5 or later
  • ExpressionEngine 1.6 or later

Bundled Fieldtypes

FieldFrame comes bundled with five useful fieldtypes:

  • FF Radio Group
  • FF Multi-select
  • FF Select
